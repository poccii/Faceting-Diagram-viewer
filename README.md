Faceting Diagram Viewer


This is a Faceting Diagram Viewer featuring a conversion function for Jambpeg machine. 
It allows you to import and view .asc files generated for GEMCAD, enabling seamless visualization of your faceting designs.

Installation
To run this tool, you will need a Python 3.11 (or higher) environment.

1. Clone the Repository
Open your terminal, navigate to your preferred directory, and clone this repository:


git clone https://github.com/poccii/Faceting-Diagram-viewer.git
cd Faceting-Diagram-viewer


2. Install Dependencies
Install the required packages by running the following command:


pip install -r requirements.txt


3. Launch the Application
Once the installation is complete, start the viewer with the following command:


python main.py
